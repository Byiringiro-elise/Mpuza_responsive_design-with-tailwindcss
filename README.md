# Mpuza - Professional Networking Platform

Mpuza is a responsive professional networking platform designed to connect job seekers with employers, primarily focused on the African market. Similar to LinkedIn, it enables users to build professional networks, browse job opportunities, and manage their career profile.

## Features

### User Features
- **Profile Management**: Personal profile with profile picture, bio, and professional details
- **My Network**: View and manage professional connections
- **Notifications**: Stay updated with network activity, job alerts, and messages
- **Job Alerts**: Receive notifications about new job opportunities
- **Courses**: Access professional development courses

### Company Features
- **Company Pages**: Dedicated pages for companies to showcase their brand
- **Job Postings**: Post and manage job vacancies
- **Follower Management**: Track company page followers and visitors

### Job Seeker Features
- **Job Listings**: Browse available positions from various companies
- **Job Details**: View comprehensive job descriptions, qualifications, and requirements
- **Favourite Jobs**: Save interesting job postings for quick access
- **Application Tracking**: Monitor job applications and interview status
- **Profile Views**: See who has viewed your profile

## Technology Stack

- **HTML5**: Semantic markup structure
- **CSS Framework**: Tailwind CSS for responsive styling
- **Icons**: Font Awesome 6.4.0
- **JavaScript**: Vanilla JavaScript for interactivity
- **Design**: Mobile-first responsive design

2. The page is fully functional with sample data
3. No build process or server required

## Customization

### Colors
Modify the CSS variables in the `<style>` section to change the color scheme:
```
css
.header-gradient { background: linear-gradient(to right, #e74c3c, #c0392b); }
.btn-gradient { background: #2f80ed; }
.company-red { background: linear-gradient(135deg, #e74c3c, #c0392b); }
```

### Content
- Replace placeholder images with actual user/company images
- Update sample job postings with real data
- Modify company information as needed
